# v0.7.7 Technical Design

## Scope

Fix the empty-workspace layout so the bottom status bar feels lighter when no document is open.

## Expected Behavior

1. When no file is open, the bottom status bar should use a more compact height.
2. When a document is open, the current bottom status bar layout should stay unchanged.
3. The change should work across all bundled themes.

## Implementation Notes

1. Mark the empty-workspace state in the shell runtime DOM.
2. Apply compact bottom-bar styling only for that empty-workspace state.
3. Avoid changing the document-opened layout so the fix stays isolated.

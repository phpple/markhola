# v0.7.7 Test Document

## Goal

Verify that dragging multiple Markdown files into MarkHola opens all dropped files as separate tabs.

## Automated Checks

1. Confirm file-URL extraction keeps all valid file paths from the macOS opened-files event.
2. Confirm non-file URLs are ignored without blocking valid file URLs in the same batch.

## Manual Checks

1. Build the app.
2. Launch MarkHola.
3. Drag one Markdown file into the app window.
   - Expected: the file opens normally.
4. Drag two or three Markdown files into the app window together.
   - Expected: every file opens.
   - Expected: each file appears in its own tab.
5. Drag a file that is already open together with a new file.
   - Expected: the already-open file reuses its existing tab.
   - Expected: the new file still opens in a new tab.

# v0.7.7 Test Document

## Goal

Verify that the bottom status bar becomes more compact when no file is open, without affecting the opened-document layout.

## Manual Checks

1. Launch MarkHola with no document open.
   - Expected: the bottom status bar looks shorter and lighter than before.
2. Open a Markdown document.
   - Expected: the normal bottom status bar layout appears.
3. Close the last open tab.
   - Expected: the compact empty-workspace bottom status bar returns.
4. Switch themes while no file is open.
   - Expected: the compact empty-workspace bottom status bar remains consistent across themes.
